## Assembly Guide
1. **Prepare Components**
   - Deburr all edges
   - Clean bushings with isopropyl alcohol

2. **Linkage Assembly**
   - Press-fit bushings (0.05mm interference)
   - Install connecting rod with 0.1mm grease gap

3. **Final Checks**
   - Verify 60mm stroke
   - Test force with load cell
